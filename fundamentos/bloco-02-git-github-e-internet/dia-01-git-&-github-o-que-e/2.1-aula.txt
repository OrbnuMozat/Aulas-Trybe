Git serve para controle de versão. Ajuda quando mais de uma pessoa trabalha no mesmo arquivo.

É fundamental saber sobre GIT para trabalhar em equipe.
GitHub é uma plataform,a para hospedar arquivos GIT.

Merge e Branch

    Branches - Versão independente do código
        Branche master - "versão master"
    Merge - une branches diferentes na branch master
    
Autenticação no GitHub
    ssh - não precisa de nome e chave de acesso
    https - utiliza certificados digitais
        Tem que criar um toquem de acesso pessoal para caso de perda de senha
        
    Trybe recomenda uso do SSH
    
Git
    add - adiciona arquivo modificado (Add file contents to the index)
    commit - liga alteração à branch
    branch - pode criar mais branch (serve para testar alterações)
        -D - serve para apagar uma branch
    checkout - serve para entrar em outra branch
        -b - cria branch e mudapra ela
    merge - une a branch
